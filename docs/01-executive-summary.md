# DoubtMaster AI — Executive Summary

## Name: **DoubtMaster AI**
## Tagline: *"Samjho, Sirf Answer Mat Dekho"* (Understand, Don't Just See the Answer)

---

## Vision
India's #1 AI-powered homework and exam preparation platform that doesn't just give answers — it ensures students **understand**. Built for CBSE, ICSE, State Boards, JEE Main/Advanced, NEET, and NCERT (Class 6–12) with native Hindi + 10 regional language support.

## Problem
Indian students spend ₹15,000–₹1,00,000/year on coaching yet struggle with daily homework doubts. Existing apps (Doubtnut, Photomath, BYJU'S) either:
- Limit free usage (5–10 questions/day)
- Have poor Hindi/regional language support
- Drop accuracy on JEE/NEET advanced problems
- Fail on handwritten equations
- Don't track learning progress
- Are bloated (100MB+ apps) unusable on budget Android phones
- Simply give answers without ensuring understanding

## Solution
DoubtMaster AI is a **mobile-first**, **sub-5MB** homework solver with:
- **Unlimited free NCERT solutions** (no daily caps)
- **99.5%+ accuracy** on Indian curriculum patterns
- **"Learn Mode"** — hides final answer until student explains the concept back
- **Hindi + 10 regional languages** natively (not Google Translate)
- **Offline mode** for rural/low-connectivity areas
- **Teacher Dashboard** — schools can track student usage with anti-cheating watermarks
- **JEE/NEET Deep Solve** — step-by-step with concept tags from PYQs

## Tech Stack
- **Frontend**: React Native (mobile) + Next.js 15 (web)
- **Backend**: Node.js + Express + PostgreSQL + Redis + Supabase
- **AI Pipeline**: GPT-4o Vision + Claude 3.5 + fine-tuned Llama 3.1 (Indian curriculum)
- **Infra**: AWS Mumbai (ap-south-1) + Vercel Edge + CloudFront CDN
